**********
Indicators
**********

Some tools to compute financial indicators. Currently contains the Bollinger Band (:func:`~fynance.features.indicators.bollinger_band`), Commodity Channel Index (:func:`~fynance.features.indicators.cci`), Hull Moving Average (:func:`~fynance.features.indicators.hma`), Moving Average Convergence Divergence (:func:`~fynance.features.indicators.macd_hist`, :func:`~fynance.features.indicators.macd_line`, :func:`~fynance.features.indicators.signal_line`), and Relative Strenght Index (:func:`~fynance.features.indicators.rsi`).
